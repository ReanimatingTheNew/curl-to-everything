在线演示

体验将curl命令转换为JSON的过程，下面是一个简单的演示示例

输入 (curl命令)

curl -X POST https://api.example.com/data \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer token123" \
  -d '{"name":"John","age":30}'
→

输出 (JSON格式)

{
  "method": "POST",
  "url": "https://api.example.com/data",
  "headers": {
    "Content-Type": "application/json",
    "Authorization": "Bearer token123"
  },
  "data": {
    "name": "John",
    "age": 30
  }
}

API文档

我们提供简单易用的API接口，可以集成到您的应用程序中进行自动化curl命令解析和转换

接口地址

POST https://curl-to.com/api/convert

请求格式

{
  "curl": "your curl command here"
}

响应格式

{
  "method": "POST",
  "url": "https://api.example.com",
  "headers": {
    "Content-Type": "application/json"
  },
  "data": {
    "key": "value"
  }
}

示例代码

fetch('https://curl-to.com/api/convert', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        curl: 'curl -X POST https://api.example.com -H "Content-Type: application/json" -d \'{"key":"value"}\''
    })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
import requests
import json

url = "https://curl-to.com/api/convert"
payload = {
    "curl": 'curl -X POST https://api.example.com -H "Content-Type: application/json" -d \'{"key":"value"}\''
}
headers = {
    "Content-Type": "application/json"
}

response = requests.post(url, headers=headers, data=json.dumps(payload))
print(response.json())
curl -X POST https://curl-to.com/api/convert \
  -H "Content-Type: application/json" \
  -d '{"curl": "curl -X POST https://api.example.com -H \"Content-Type: application/json\" -d \'{\"key\":\"value\"}\''"}'
